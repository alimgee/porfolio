Projects & Commisions

DevAssist - Full Stack Project using Python and Django

This is my final project from the Code Institutes 'Full Stack Software Development' course. This fully respnsive website intends to mimic a forum type environment where registered users can raise technical queries and respond to other users queries, with an ecommerce option to donate funds to the site creators. Python, Django, Bootstrap and Stripe technologies are used to achieve this. The marks from this project contributed to the receipt of a diploma level award. The site can be accessed from https://dev-assist.herokuapp.com/ . The code base is viewable on my github.

Bookbites - Project in Python and Flask

This is my 3rd project from the Code Institutes 'Full Stack Software Development' course. This fully responsive site mimics a book review site where users can register and log in to add reviews and also like other users reviews with a 'bookbite'. It was built using Python, Flask and a MongoDB database. The marks from this project contributed to the receipt of a diploma level award. The site can be accessed from https://book-bites.herokuapp.com/ . The code base is viewable on my github.
